# Tata-Motors-Financial-Analysis
3-year financial analysis of Tata Motors (NSE: TATAMOTORS) covering consolidated P&L, Balance Sheet, Cash Flow Statement and 18+ financial ratios for FY2023-FY2025.

Key findings: Net profit swung from -₹16,955 Cr loss to +₹14,113 Cr profit.

Debt-to-Equity fell from 5.59x to 1.19x through aggressive deleveraging. 

Free Cash Flow grew to ₹71,522 Cr in FY25. Built using Microsoft Excel with live formula referencing.
